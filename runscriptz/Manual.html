RunScriptz

A simple docker that let you run your script located in one folder, faster than loading in Scripter. Put all your python script for krita in one folder Point that folder to the Script it let you run them with a click or hotkeys. 1-Install Script through Krita Plugin Manager 2-Start the docker Settings>Dockers>RunScriptz 3-Click on the button "Select Folder" to select the folder where your scripts are located 4-All scripts should show up in the list (Press Refresh to force reading the folder again) It support subfolders 5-You can now run your scripts by double clicking on them or by using the hotkeys you assigned to them 6-Assing hotkeys to your script by right clicking on them and selecting "Assign Hotkey"
