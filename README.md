## Usage

1. **Set Scripts Folder**: Click "Folder" to choose a folder containing Python scripts
2. **Run Scripts**: Click on any script or use the "R" button
3. **Assign Hotkeys**: Right-click on scripts ‚Üí "Assign Hotkey"
4. **Switch Modes**: Click the mode button to toggle between List and Button views

## Hotkey Management

- Hotkeys are saved automatically
- Right-click scripts to manage hotkeys
- Hotkeys appear in Krita's keyboard shortcuts menu
- Visual indicators show assigned hotkeys in both modes
